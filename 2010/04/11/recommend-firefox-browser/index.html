推荐使用火狐浏览器

2010-04-11

在拥有独立博客之前，也就是2008年十月份之前，我一直都在使用IE浏览器，极少使用其他浏览器。当自己建立这个独立博客后，我认识了很多人，了解的东西也越来越多，然后就看到很多同学都在使用火狐浏览器，于是我也下载了一个试用了一下，结果一下子就喜欢上火狐浏览器了，除了用网银和支付宝外，我极少使用过IE浏览器。火狐浏览器的功能本来不算强大，但它提供插件支持，这样使得火狐浏览器功能变的非常强大，完成了很多浏览器不能完成的工作，比如过滤互联网上浩浩荡荡的广告，整个网页截图，网页设计工具等等，深得广大站长和博主的喜爱。

火狐浏览器(Firefox)内置了分页浏览、拼字检查、即时书签、下载管理器和自定义搜索引擎等等功能。此外还可以通过由第三方开发者贡 献的扩展来加强各种功能，还支持多种操作系统。火狐浏览器(Firefox)一直被大家非正式的称作“FF浏览器”。下面是我一直在使用的几种插件，感觉很不错。另外，Mozilla还开发出了Mozilla Weave，其可以实现同步书签，历史记录，已保存密码，浏览器设置和标签等功能。

1. Adblock Plus：广告过滤插件。安装上之后可以屏蔽来自互联网上的很多广告，也可以自己写一些过滤规则来选择性的过滤哪些广告，当然你也可以选择在哪些站点禁用此插件。下载地址：https://addons.mozilla.org/firefox/addon/1865

2.Pearl Crescent Page Saver Basic：网页截图插件。可以截取整个网页的图片，不管这个页面有多长，有时当你浏览一个页面时，想收藏这个页面，那么你只须点击插件图标就能把这个页面以图片的形式保存下来，不必担心这个页面被删除。这个插件允许你自定义要保存图片的格式，名称，可以以标题、时间、URL等来命名，还可以选择是保存可见区域还是整个页面。下载地址：https://addons.mozilla.org/firefox/addon/10367

3.Firebug ：网页开发工具。官方介绍：Firebug 为你的 Firefox 集成了浏览网页的同时随手可得的丰富开发工具。你可以对任何网页的 CSS、HTML 和 JavaScript 进行实时编辑、调试和监控。站长博主和网页开发人员的最爱。下载地址：https://addons.mozilla.org/firefox/addon/1843

4.User Agent Switcher：切换浏览器user agent。该插件可以实现访问只有在手机上才能访问的页面，如3GQQ、手机wap站点。例如，当时你在浏览器里输入wap.baidu.com、sina.cn等手机站点时会打不开或者自动跳转到平时浏览页面，但是当安装该插件后，你可以切换一下user agent，这样就可以浏览这些站点了。下载地址：https://addons.mozilla.org/firefox/addon/59

5.FlashGot：下载文件时的插件。该插件可以让用户自己选择下载机器上已安装的下载工具下载文件。该插件还能加快下载速度。下载地址：https://addons.mozilla.org/firefox/addon/220

6.AutoProxy：官方介绍：你要翻墙对吗？你想让它聪明点：该翻墙时才翻，不要所有的网站都用代理，对吗？你还想让它再聪明一点：该不该翻墙它自己知道，不用什么事都要你跟它讲，对 吗？ AutoProxy 就是为您设计的：）。下载地址：https://addons.mozilla.org/firefox/addon/11009

此外还有很多优秀的插件，在此就不再多说了，以上几个都是我常用的插件。这是我收集的比较优秀的火狐浏览器插件(https://addons.mozilla.org/firefox/collection/chenzhiwei)，大家可以来围观一下，等遇到好插件时我会及时添加进去的。

下面是火狐浏览器所有版本的下载地址：http://www.mozilla.com/en-US/firefox/all.html

另外，这个地址是最新试用版的下载地址：http://ftp.mozilla.org/pub/mozilla.org/firefox/nightly/latest-trunk/，该版本的火狐浏览器可能会不稳定，但是里面会包含一些新的功能及新的UI界面。
