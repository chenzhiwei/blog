纪念那些我所创建的百度空间

2015-04-07

今天浏览微博看到百度空间即将关闭，多少让人有点惋惜。我是从 2006 年开始使用百度空间的，那时百度空间刚成立没多久。后来断断续续的使用了一阵子，再后来我就有了自己的独立博客了，然后就没再使用过百度空间。

第一次接触百度空间记得是在第一年高考成绩出来之后填报志愿时，那天晚上和同学去网吧上网，不知道是搜索什么的东西来着就进入了一个百度空间。我记得地址是hi.baidu.com/xinhui，里面是介绍一些电脑知识和影音推荐的内容，我还加了这个人的 MSN，交流了一段时间，他的名字叫Steven Chen，人在澳大利亚。读大学后还和这个人有一段时间的邮件来往，再后来我不怎么登录 MSN 了，他的 MSN 也不怎么在线了，然后就再也没联系过了。互联网就是这样，你可以很容易的联系上一个人，也可以很容易的和一个人失去联系。

我在百度空间还认识了两个从来没见过面的女孩，都是在大学时代认识的。其中一个已经失去联系将近五年了，另一个虽然在我的微信好友列表里，但是也有很长时间没联系了。之所以写这些，是因为我从她们那里学到了一些东西，并且还给我留有不错的印象。

百度空间以前支持自定义 CSS 样式，可以设计出自己喜欢的样式，当时吸引了不少人。估计后来的产品经理是个大傻逼，把百度空间改的不像样了，然后就落得今天这个被关闭的田地。

我的第一个百度空间名字是万叶集，还是一个 QQ 上的网友给起的名字。这个空间虽然没更新过，但是却一直存在的。直到上次百度空间改版，由于我已经忘记这个帐户的密码，没法迁移，于是我在百度的第一个百度空间就这样完蛋了。当时我还是李宇春的粉丝，空间已经不复存在，截个百度帐户的个人主页以示纪念吧。

第二个百度空间是在大一（2007）时注册的，当时还是自己给自己起的一个英文名，太装逼了，没多久就被我废弃了，同样截个百度帐户的主页做纪念吧。

第三个百度空间在 2008 年初注册的，当时百度即时通讯工具百度Hi刚刚上线，于是我就注册了一个新的百度帐户来使用。还好，这个帐户和空间的密码我一直都记着，并且百度空间改版迁移时我也进行了迁移，空间里的东西都还保留着。最近一次更新是在 2010年04月18 日，之后就一直没怎么使用，里面的内容就是一些转摘和一些蛋痛的内容。之所以没怎么使用百度空间是因为我在 2008 年有了自己的独立博客，接着的时间都是围绕我的独立博客展开的。截图纪念。

第四个空间是在 2012 年注册的，这空间里没有什么内容，只是占个坑。本来我是想把自己转载的内容都发到这个空间里呢，结果发现 Lofter 更合适，并且 Lofter 还支持绑定独立域名。顺便给我的 Lofter 转载基地做个广告：http://read.chenzhiwei.cn

写到这里，我对自己都很无语了，注册这么多东西有毛用啊。。。其实，我注册的博客可不只这几个，还有很多很多。我把时间都花费在了折腾博客系统上面了，而写的东西却少之又少，有点本末倒置了。

话说来回，还是有一个自己域名下的独立博客更靠谱，不会被消失。最开始一直用 Wordpress 博客系统，虽然我是做技术的，但是感觉维护一个 Wordpress 博客还是太麻烦，于是我就转向了这个 Octopress 博客系统，直接用 Git 进行管理，不用再经常备份博客主题或博客数据库了，真爽。

关于邮箱，其实我也建议放在自己的域名下面，上次那个号称「无限容量」的免费「终生邮箱」的 Yahoo 中国邮箱不就关闭了么。
