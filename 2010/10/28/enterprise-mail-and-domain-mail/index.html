汇总一下免费企业邮箱和域名邮箱

2010-10-28

如今，越来越多的人拥有了自己的域名，也有越来越多的人使用自己域名做后缀的邮箱，类似于这样的：user@domain.com，使用自己域名作为邮箱后缀的好处我就不多说了。我是在2008年10月份接触域名邮箱的，那个时候我注册的是google apps mail，用的是CN域名，当时google是不允许CN域名注册google企业应用套件的，但是在网上看到一篇文章，写的是将域名后的cn大写就可以了，填写国家时写中国香港就行。我就是在这时注册的google企业套件，一直到现在我仍在使用。前些日子，google又将企业应用套件帐户和google帐户做了个整合，这样一来，使用google企业应用套件帐户就能使用google帐户里的大部分功能了。

早在2006年11月1日时网易就推出了域名邮箱公测版，但是在2006年12月29日就关闭了。在随后的一段时间里，国内的一些公司也陆续推出了企业邮箱（域名邮箱），也就是允许使用自己域名作为邮箱后缀。当初搜狗也推出了域名邮箱一段时间，很快也禁止新用户注册，当时的公告好像是只有在搜狗上注册域名才允许使用他们的域名邮箱。现在国内的很多公司都推出了免费企业邮箱（域名邮箱），包括：腾讯、网易、搜狐、新网互联等。国外的比较熟悉的免费企业邮箱就属google企业应用套件和微软Live Domain了。其中除了搜狗和新网互联的免费企业邮箱我没使用过外，其他的我都用过。

1. google企业应用套件标准版

http://www.google.com/apps/

官方介绍：使用 Google 企业应用标准版，您的俱乐部、家人、运动团队或群组可以轻松地进行在线通讯和协作，而且完全免费。

产品包括Gmail，google协作平台，google日历，google文档，Gtalk等一系列google产品。google企业应用套件邮箱是通过https加密访问的，使用起来绝对安全。现在google企业应用套件帐户和google帐户已经互通了，使用google企业应用套件帐户还能够使用google picasa相册，google 阅读器，blogger等一些优秀的应用。对国内用户来说，注册google企业套件主要使用的功能是Gmail、Gtalk、google 日历，google 阅读器等，其他的google产品在国内访问是有困难的。google企业应用套件标准版中邮箱的大小是7G，一般就够你用一阵子了，不用但心容量不够。另外，google企业应用套件标准版的用户数量是限制在50个用户，以前可以申请到最多2000个用户，现在我的用户数量就是500人，一般用不了这么多。来这里了解更多关于google企业应用套件的信息：http://www.google.com/apps/ 。

2. 腾讯企业邮箱

http://exmail.qq.com/

官方介绍：腾讯企业邮箱是一种邮件托管服务。企业仅需要将自己的域名进行简单的设置，就能够创建属于自己的邮局。并管理以自己域名结尾的邮箱帐号，把它们分给员工使用。

产品包括基本的企业邮箱、免费短信提醒、企业网盘和企业地址本等。腾讯企业邮箱是全程SSL安全接入，安全性方面也行。腾讯企业邮箱目前支持帐户数量是200个，如果有需求可以向他们申请增加用户数量。另外，腾讯企业邮箱支持绑定QQ号，可以从QQ客户端面板直接进入企业邮箱，相当方便。如果你只是一个普通用户，对邮箱没什么特别需求，那么腾讯企业邮箱肯定是首选。更多信息请查看腾讯企业邮箱官方网站：http://exmail.qq.com/。

3. 网易免费企业邮

http://ym.163.com/

官方介绍：网易免费企业邮是网易公司推出的一项免费的、个性化邮件服务，只要用户拥有域名，就能够创建以用户域名作为邮箱后缀的邮箱。

网易免费企业邮相比google企业应用套件和腾讯域名邮箱来说，功能不多但比较专一。网易邮箱品牌很多人都是知道的，国内邮箱用户数量也是相当庞大的。网易免费企业邮默认用户数量也是200个，由于功能专一，所以界面比校简洁。更多信息请查看网易免费企业邮官方网站：http://ym.163.com。

4. QQ域名邮箱

http://mail.qq.com

官方介绍：域名邮箱是QQ邮箱提供的一项个性化服务。如果您有域名，您就可以使用自己的域名作为邮箱后缀了。同时，您还能更自由的选择您的帐户名和邮箱LOGO。

我之所以称其为QQ域名邮箱而非腾讯域名域名邮箱是因为QQ域名邮箱和QQ邮箱是使用的同一个邮箱，是一个邮箱多个地址。现在腾讯已经推出免费企业邮箱了，有谁还会用QQ域名邮箱呢。这里就不多做介绍了，详细信息请查看这里：http://domain.mail.qq.com/zh_CN/htmledition/dm_learnmore.html。

5. 微软Windows Live Admin Center

http://domains.live.com/

官方介绍：如果您担任团队教练、管理俱乐部，或者您正需要一个供家人使用的个人联机空间，那么您可以使用 Windows Live 管理中心为您的群设置个性化 Windows Live 服务。成员可以获得各自的自定义电子邮件地址，并且可以聊天、发送电子邮件以及与群内的其他成员共享文档和照片。

微软Windows Live Admin Center以前叫Live Domain，微软这个东西感觉并不怎么样，我注册试用过，没什么意思。和hotmail一样，不建议注册。如果你有兴趣也可以去其官网了解一下：http://domains.live.com/。

6. 搜狐企业邮箱

http://mail.sohu.net/free/

官方介绍：搜狐企业邮箱以功能强大、稳定、产品易用著称，贴近企业管理实用功能，让您感受到国内最专业电子邮件服务为您带来的安全、告诉、稳定的网络办公新生活！

搜狐企业邮箱是最近刚刚推出的，我还没注册试用，对此我没什么可说的。其实作为一个普通用户，腾讯企业邮箱和网易免费企业邮就够了，如果你喜欢搜狐产品的话可以注册使用。详细介绍：http://mail.sohu.net/free/。

7. 新网互联的云邮免费版企业邮箱

http://www.dns.com.cn/

新网互联的这个免费版企业邮箱支持的用户个数要邮箱容量都不多。没什么优势，并且好像还是免费一年，一般情况下是没有人注册的。详细信息：http://www.dns.com.cn/main/public_doc.php?doc=G5/index。

以上就是我所知道的几个免费的企业邮箱（域名邮箱）了，个人认为，如果你经常使用QQ，并且不用邮箱做一些特别的事情，那么建议你使用腾讯企业邮箱，当然，网易免费企业邮也是不错的选择，至于选哪一个就要看你自己了。如果你对邮箱的安全性要求比较高，那么建议你注册google企业应用套件。至于其他的几个企业邮箱（域名邮箱）就看个人爱好了。我现在正在使用的企业邮箱有google企业应用套件标准版、腾讯免费企业邮箱和网易免费企业邮箱，我主要登录的邮箱只有一个，将其他邮箱的邮件设置转发了。
