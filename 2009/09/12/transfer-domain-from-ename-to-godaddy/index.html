成功将.net域名转移到godaddy

2009-09-12

前些日子我获得了个.net域名，这个.net域名原本在易名中国上面，当我仔细查看易名中国的服务条款时发现，.net域名续费居然是100元人民币每年，像我这样的穷学生哪能每年拿出来100元来养一个域名呢。在早些时候也看了很多文章，很多人都把域名转移到国外去了，于是我也跟风把域名转到国外了，国外的域名注册商中godaddy与name不错，而我在godaddy上已经注册了个域名了，于是就将域名转移到了godaddy上面。

易名中国总体上还可以，索取域名转移码要比万网与新网方便的多。易名中国的域名转移码是靠手机短信获取的，开通手机短信获取转移码需要先通过易名国的实名认证。因为我在早些时候已经把自己的身份证扫描后放在了邮箱里，所以很轻松的我就通过了易名中国我实名认证，不过实名认证要花钱的，认证一次5元钱。然后就是开通手机短信提醒服务了，这个短信提醒是用积分来获取的，因为我没有积分，所以就拿金钱来买了，1元10条。我按照提示输入手机号，填写一下手机获取的验证码就开通了。不过让人无奈的是，易名中国的手机验证码是在短信通知行列的，接收一次验证码就会扣除一毛钱，我在整个域名转移过程中一共收到了三条手机短信，花了我三毛钱。分别是开通手机短信提醒时一条，域名获取转移码确认是一条，得到域名转移码一条。易名真是黑，不放过任何能够剥削用户的机会。

在开通手机短信获取域名转移码时，得先下载个他们提供的表单，然后打印出来把相关内容填写上去，再扫描到电脑中连同身份证扫描图片一并以电子邮箱的形式发送至他们指定的邮箱。大概两天左右，接到了他们打过来的电话，确认一下用户身份，然后就给开通了短信获取域名转移码功能。不过，你也可以直接将易名中国提供的表单打印出来，填写好相关信息后连同自己的身份证复印件一并邮寄到他们公司，不过这样做效率不高。

在一切就绪后，我就开始查看域名转移至godaddy的步骤了，等到熟悉之后，我就从易名中国那里获取域名转移码。在获取域名转移码时有个小插曲，易名中国的短信获取域名转移码平台不支持firefox浏览器，而我一直都在使用firefox浏览器，当时我见情况不妙就给易名中国客服打了个电话，告知他们情况后，他们也不知道怎么回事，因为他们那边没有我短信获取转移码的记录，我都快无奈死了。过了一会，一个客服MM打来电话询问情况，正在和她聊着的时候，我突然想到了用IE浏览器试一下，给客服MM说明情况后，她就没有挂断电话一直等把把事情搞定后才挂上电话。于是，域名转移码我算是拿到了。

购买giftcard。在QQ群里，一位同学告诉我一个淘宝上的卖家，我加他旺旺号之后就聊了起来，他们购买giftcard一个小时就差不多了，于是我们很快就完成了交易。不过等了四个多小时，一直没有收到giftcard，当时我就和他说明情况，他说72小时内收不到就退款，于是我就忍着等到了第二天，谁知道等到第二天中午我还是没有收到giftcard，在与他协商后，决定退款，他提出的条件时，我收到giftcard时，把giftcard给他，他就给我退款。不过，第三天时他就同意退款了，具体情况我也不清楚，反正一直到现在我也没有收到他的giftcard。后来我发现，126邮箱不能接收国外公司自动发送的电子邮件了，这大概就是我收不到giftcard的原因吧。

在idcspy那里买到giftcard后，马上就在godaddy上完成了交易。完成交易进行域名转移界面后会提示让我输入一个transaction id 和 secure code ，我当时还很好奇，不知道是怎么回事。在网上查了一下，又问了几个网友，才知道这个ID与CODE是发到域名管理员邮箱里，而我的域名管理员邮箱是126的，一直没收到邮件。我又等了一天，还是没收到，12个小时过去了，我重发了N次还是没有收到，后来在网上查了下，可能是因为国内邮箱的原因，于是我将域名管理员邮箱改成yahoo邮箱后马上就收到了ID与CODE。填写好之后会出现一个等待原注册商确认阶段，一般5天左右，5天内如果原注册商不做处理的话，域名会自动转移成功。我是刚刚收到域名转移成功邮件，一共用了6天。

其实在国外注册域名与将域名转移到国外很容易，没有想像中的那么复杂。域名转到国外的好处是可以随便转移注册商，不会因为网站上有不和谐的言论而被暂停解析，鼠标轻轻点击一下就能收到域名转移码，域名续费价格也较国内注册便宜。

最后我告诉大家一件事情，如果你有国外网友或者注册国外网站的一些服务的话，请使用国际邮箱，如Gmail,yahoo mail，hotmail等。国内邮箱有时会收不到国外的电子邮件，在我这个域名转移过程中就出现了两次不能收到电子邮件的情况。在之前的一个月我126邮箱还能收到godaddy发的电子邮件呢，而这几天却收不到了，可以是因为祖国60岁生日的原因吧。
