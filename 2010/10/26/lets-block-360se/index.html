我的博客已经屏蔽了360浏览器

2010-10-26

靠做流氓软件起家的360董事长周鸿祎早就恶名在外，做人不能太周鸿祎！已经成为互联网一句口头禅！本人非常不喜欢360系列软件及周某某，一个周流氓领着一群XX开发了一系列流氓软件，并且还成功愚弄了某国的N多初级网民，稍微有点网龄的高级网民是不会使用360系列软件的。

周鸿祎可谓树敌无数，从3721 时期与CNNIC的宿怨、离开雅虎后与阿里巴巴的官司，到杀毒软件领域与金山、瑞星、江民、卡巴斯基等传统杀毒软件商撕破脸，今年以来与求伯君、可牛的傅盛爆发口水大战，与百度在桌面领域的新仇旧恨一并对决，得罪四大门户和凤凰网，最近又和小企鹅干上了，可谓彪悍之致！试想一下，我们不用360系列软件，还有很多可以代替的软件，比如：金山卫士、QQ电脑管家、可牛杀毒等，但是如果我们不用QQ的话，我实在想不出还有什么软件可以和QQ相比的。当初阿里巴巴马云选择收购phpwind而不收购discuz的一个原因就是周鸿祎投资了discuz，马云不想再与周鸿祎有任何关联，由此可见360老总周鸿祎的人品的确不怎么样。

抵制360系列软件，从自己做起。我自己已经不用360系列软件很长时间了，并且我也劝过很多身边的人，让他们也不使用360系列软件。最近，我在看博客流量统计时，发现我博客上还有一部分访客使用的是360浏览器，于是我就从网上搜索了一下，找到了一篇屏蔽360浏览器的文章，找到相关代码，加入了自己的博客中。现在360浏览器已经没法访问我博客了。

我希望我认识的人，和访问我博客的博友都能像我一样，主动屏蔽360浏览器。以下是屏蔽360浏览器的代码。

屏蔽360浏览器访问的php源码：

<?php
    $useragent = $_SERVER['HTTP_USER_AGENT'];
    if(strstr($useragent,"360SE")){
        die("系统检测出来你使用了360流氓浏览器,请先卸载或改用ie（或firefox）等再行访问本站，谢谢合作！");
    }
?>

屏蔽360浏览器访问的Js脚本源码：

<script language="javascript">
UA=navigator.userAgent;
if (UA.indexOf('360SE')>0){
    alert("本站拒绝安装了流氓软件360的用户访问，想要舒心浏览本站请卸载360相关“安全软件”，谢谢合作！");
    document.execCommand("stop")
}
</script>

linux+nginx服务器屏蔽360浏览器访问的代码:

if ($http_user_agent ~ (360SE) ) {
    return 404;
}

注：本文参考网站屏蔽360浏览器访问的代码，有改动。
