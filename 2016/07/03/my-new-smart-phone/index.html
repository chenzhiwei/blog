新入手了一部 Android 手机

2016-07-03

这两年一直在关注新的 Android 手机，国内国外的手机都有关注。但是一直都没找到完全符合我要求的手机，包括我两周前入手的这部一加手机 3 代，仍然没能完全符合我的所有期望。

从 2014 年夏天开始，我就一直想换一部手机，但是一直没有完全符合我期望的手机，我对手机的要求也越来越多，总结一下：

  • 原生 Android 或接进原生 Android
  • 5 寸左右，单手操作良好
  • 机身正面不能有任何按键，虚拟键内置于屏幕中
  • 指纹识别
  • 扬声器不能放置在机身背部
  • 接口为 USB 3.0 Type-C

然而我目前入手的一加手机 3 代有两点没能满足我的要求：尺寸太大和正面指纹识别按键，其实这个手机接口是 USB 2.0 Typc-C。但是它却有一个让我非常愿意去入手的理由，开放手机底层源代码，也就是说别的第三方 Android 系统可以很轻易的移置上去，当然我比较喜欢的 Ubuntu Touch 未来也极有可能移动上去。

5.5 的尺寸对于我来说确实是有点偏大，单手操作不方便。而关于机身正面的电容键，在我刷入 Oxygen OS 之后就直接禁用掉了，也就是说正面目前只一个指纹识别键还在使用，但是仅做指纹识别使用，屏幕内置的三个键使用起来还是挺方便的。

我入手手机之后的第一件事就是下载并刷入国际版的 Oxygen OS 系统，因为这个才像 Android ，国内的氢 OS 就是一坨屎，超级难用。在国内全天开启 Shadowsocks 并使用原生 Android ，体验还是不错的，并没有太多的不舒服的地方。

对了，这部手机是我入手的第三部智能手机，第一部是摩托罗拉的 Defy(MB525) ，第二部是小米手机 2 代。
