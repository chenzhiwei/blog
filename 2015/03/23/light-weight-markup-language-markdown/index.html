轻量级标记语言 Markdown

2015-03-23

Markdown 是一种轻量级标记语言，目标是让人们以「易读易写」的纯文本格式来编写文档，然后将其转换成 HTML 文档。

我也是工作之后才接触 Markdown 的，当我第一次接触时就已经非常喜欢它了。只需要记住几个简单的规则写一篇纯文本文档，然后轻易的转换成格式优美的富文本文档，这绝对是写文档的利器。

对于程序员来说，写程序才是主业，然而很多时候却被写文档折腾的痛苦难受。有了 Markdown 以后，再也不用理睬各种各样的富文本编辑器了，尤其是让头痛的 MS Word 。用 Markdown 写出来的文档无论是纯文本还是富文本，可阅读性都很高，现在几乎所有的开源软件的文档都以 Markdown 形式来发布了。

对于一些编辑或码字的人来说，Markdown 也是不可缺少的。在码字的时候顺手添加几个特殊字符，就可以实现分级标题、内容引用或加粗显示，然后继续码字，不需要回过头来给内容设置分级及强调引用等操作了。

Markdown 的语法中使用的特殊字符都是精挑细选的，其作用一目了然，基本上一次就能记住它们的用法了。当然也有一小部分不太常用的，可能需要多用几次才能记住，但无论如何这都要比用鼠标在富文本编辑器的按钮和内容区来回切换点来点去强的多。

当然 Markdown 也不是万能的，HTML 的很多语法都无法用它来实现，好在 Markdown 完全兼容 HTML 语法。如果你想使用 HTML 中非常特殊的用法，你可以直接在 Markdown 里写原生 HTML 格式的内容，不过我想你应该会很少遇到这种情况的。

目前有很多项目代码管理系统和一些写作平台都支持 Markdown 格式，以后会有越来越多的平台支持 Markdown 。我觉得将来网络上会到处出现 Markdown 的身影，会简单的书写及阅读 Markdown 是很有必要的，我博客现在就是完全用 Markdown 书写的。

另外，如果你使用 Chrome 浏览器的话，推荐你使用这个 Markdown Editor。
