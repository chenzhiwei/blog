网站建设算是告一段落了

2009-06-03

从三月份开始做的网站，一直到今天才算是全面做成了，之前没有做过这样的网站。都是弄一些个人博客之类的东西，比较简单，有些功能实现不了也没事，因为自己的博客，自己做主。

关于网站，这次做网站，我选择的是南方数据的模板，刚开始看着还差不多，慢慢的我发现，他们这个公开的系统，后台有很多功能都不能实现，很多功能都是我自己看着asp代码加上去的。一直到现在网站的静态功能我都还没有实现。最终，我决定使用动易网站管理系统的后台，然后改了一个南方数据的前台。不过现在看起来一点也不想南方数据的，因为我已经做了很大的更改了，考虑到网站信息量较大，我就把新闻搜索栏目放在了首页显眼位置。界面总体上来讲还说得过去，但是首页布局有些不合理，不太美观。后台那里也有一点小问题，就是在ewebeditor里面上传的图片不能预览，这是因为我把后台的ewebeditor给换掉的缘故，我重新写了一下后台，所以有些地方与前台结合的并不是很好。后台还有很多功能，比如添加友情链接，更改主题，更改网站配置信息等，都能实现，不过我把这些功能禁用了，因为一般情况下都用不到这东西。

关于ASP.NET，我们现在开着C#课，而我因为对C#不感兴趣而没有认真听过，在不得已情况下，我从网上下载了个开放式实验在线预约系统，因为网上ASP的不要求，所以就下载了个ASP.NET的，我对ASP.NET又一点都不了解，于是我从昨天下午一直忙到晚上十点半都是研究ASP.NET的运行环境，在网上查找了很多资料，又在QQ上问了很多asp.net技术牛人，让他们用QQ控制我机子来教我使用IIS运行ASP.NET程序，我非常感谢他们，“感覺”是教会我怎么在IIS下运行ASP.NET程序的，因为我使用的机子上已经安装了VS了，所以就不用再装其他东西就能在IIS下运行了，而周沛是教会我怎样配置IIS来支持ASP.NET的，他教会了我在XP下和win 2003 server下配置asp.net环境，他是把他2003 server的配置截图发给我的，非常的感谢他，没有他，我现在还不会配置ASP.NET环境，更不能在win 2003 server 下来写这篇文章。

关于ewebdtitor，在做这个网站之前我不知道ewebeditor是什么东西，但是现在我知道了，因为我在做这个网站时用到了ewebeditor，它的默认上传文件大小太小了，我呢，就阅读了一下ewebeditor的代码，把最大上传文件大小调高了10多倍，但是现在仍然有个问题没有解决，就是ewebeditor的上传图片预览还是没有做好，我把后台的ewebeditor给换掉了，所以这个预览功能一直没能实现，不过，不影响上传与使用。

关于IIS服务器，IIS是微软的一个产品，说它牛B它牛B，说它不牛B它也不牛B，牛B是因为它简单易用，不牛B是因为，在使用过程中老是出现一些莫名其妙的错误，让人不知所措，很难找到原因。有一次是IIS上午还用着好好的，下午网站就不能访问了，让我用了一个晚上，共4个多小时才搞定，我在网上找了很多解决办法，比如重新安装IIS，启用一些服务什么的，哎，统统都不管用。最终当我决定要离开的时候，突然在百度知道上看到，有可能是因为微软的补丁（KB939373）的原因。我按照上面所讲，把这个补丁卸载掉之后，再一重新调动服务器，一切都搞定。今天下午学校网络中心那边把实验室的一台电脑映射成了服务器，刚开始用的是XP系统，还不到两小时，网站就崩溃了，提示“目前访问网站的用户过多”，很是不理解，在网上找了很多方法，但都无济于事，最后又让同学(东跃影院的站长)在QQ上使用远程把那个默认的允许访问人数从10人加到了10000人，但这只当用一会，过了不到20分钟又不行了。最终还是让他把他用的win 2003 server给我发过来，我这边重新安装了一个win server 2003服务器才算是把问题解决了。我自己从来没有用过2003 server ，对它一点也不了解，还是让同学远程协助帮忙整的。现在，一切问题都已经解决了。

我现在要说的是，如果不在情非得已的情况下，请尽量不要使用微软的产品，因为微软的产品很那个的。用过微软产品的同学应该都知道吧，VS.NET安装是多么的让人无奈吧，IIS一出问题也是很难解决的。也不知道怎么回事，为什么微软自己的产品会和自己的产品冲突，微软的产品总有打不完的补丁，找不完的漏洞。

这段时间，我是真的太忙了，JSP课程已经有连续三个周没有上了，老师给我们布置的项目我现在还没有开始，而有些同学都已经完成了，我的数据库项目也没有开始做，一切都没有开始。我想从明天开始，一切都会回归正常，我的生活也将会更加繁忙，因为我要完成别的同学早就开始做的项目。六级考试眼看着也就要来临了，而我现在一点还都没有准备。网络课周五就要考试了，我现在什么都还不懂。一切都是这么我紧张，生活总是这么忙碌。朋友说，这样的生活才充实，不管理与否，我已经在过着这样的生活了，我也从这种生活中学到了很多东西。在此我还要请求我的老师们原谅我最近的旷课，我从上大学到现在可以说都是一个很遵守纪律的学生，不旷课，不请假，每节课都按时上，虽然我有时不听课，但我很少影响到别人。

很久没有写这么长的东西了，今天整了个通宵来弄这个网站和配置win 2003 server，经过多次测试，一切都很正常。所以我才写这些东西来记录下我在2009-06-03到2009-06-04号的夜里没有休息，其实我上周六就整一个通宵了，一直都没有休息，现在真有点佩服我的精力。现在已经接近四点了，我却还没有感觉到特别困，真的感觉自己很了不起。对了，忘说了一点，今天夜里还有一个好哥门一直陪我到现在呢，呵呵，也许在这个时候才能看出什么是真正的友谊吧。
